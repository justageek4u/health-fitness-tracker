# Health and Fitness Tracker V3 Starter

This is a Vite + React starter package prepared for Vercel deployment.

## What is included now
- Section 1 shell/navigation
- Section 2 dashboard
- Mobile hamburger navigation
- Current default foods: Fairlife, White Rice, Cottage Cheese
- Local browser storage

## Not finished yet
The remaining tabs are still placeholders in this starter package:
- Log
- Food & Weight Entry
- Exercise & Sleep Entry
- Foods / Meals
- Workouts
- Goals
- Export / Backup
- How-To

## Local run
1. Open a terminal in this folder
2. Run `npm install`
3. Run `npm run dev`

## Vercel deploy (GitHub route)
1. Create a GitHub repo and upload this folder
2. In Vercel, choose **Add New Project**
3. Import the GitHub repo
4. Vercel should detect this as a Vite app automatically
5. Click **Deploy**

## Vercel deploy (CLI route)
1. Open a terminal in this folder
2. Run `npm install`
3. Run `npm install -g vercel`
4. Run `vercel`
5. Follow the prompts
